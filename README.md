# 🛒 CodeAlpha Ecommerce Store

A responsive and modern **Ecommerce Store Website** created using **HTML, CSS, and JavaScript**.  
This project provides a clean shopping interface with product display, stylish design, and interactive functionality.

---

## 🚀 Features

✔️ Responsive Design  
✔️ Attractive UI Layout  
✔️ Product Showcase Section  
✔️ Add to Cart Interface  
✔️ Smooth Navigation  
✔️ Clean Code Structure  
✔️ Beginner Friendly Project

---

## 🛠️ Built With

- HTML5  
- CSS3  
- JavaScript
